CSS styles

Inline, Internal (Embedded) and External

Inline is in HTML tag, Embedded in the head of page (by using style tag) and External is in a seperate file.(uses link tag in the head of document)

Inline is in the highest hierarchy and external is lowest.

Background image

Use cover for background size so that image would always cover the whole page

use no repeat value so that image would not repeat if the size is small
